# Vagrant Boxes
A collection of vagrant boxes to run various operating systems with the default setup.

## Prerequisites
- [VirtualBox](https://www.virtualbox.org/wiki/Downloads)
- [Vagrant](https://vagrantup.com/)

## Available Boxes

- [Windows 10](windows/10)
- [Windows 7](windows/7)

## Usage

    $ cd <box directory>
    $ vagrant up 

## Maintainers
